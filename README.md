# Park Golf Score Android

파크골프 랭킹 시스템을 Android 앱으로 제공하는 WebView 앱입니다.

## 웹사이트

https://joohyeonkim.github.io/Park-Golf-Ranking-System/

## 기능

- 파크골프 랭킹 시스템 웹페이지를 전체 화면으로 표시
- 뒤로가기 버튼으로 웹 히스토리 탐색 지원
- 시스템 바(상태바/네비게이션 바) 영역 자동 회피

## 빌드

```bash
./gradlew assembleDebug
```

## 요구 사항

- Android SDK 24 (Android 7.0) 이상
- 인터넷 연결 필요
